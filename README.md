# launcher-remote-pro (public)

Public hosting for [Launcher Remote Pro](https://jeongjihun.github.io/launcher-remote-pro/) downloads and Sparkle update feed.

- Landing page: `index.html`
- Sparkle feed: `appcast.xml`
- Release artifacts: `releases/*.dmg` + `releases/*.dmg.sig`

Source code lives in the private repo `launcher-remote-pro-app`.
